# 7DOVE-SPRAWDZANIE-PLUGIN READ BELOW:
Minecraft Autorski Plugin Do Sprawdzania Graczy, Po Wgraniu Pluginu Prosze Uzyc LP ( LuckyPerms) I Utworzyc Range Sprawdzany, Jesli Default = Ranga Gracza Prosze Ustawic Default Na Range Gracza.

# Komendy

- Komendy Przez " / " Zostaly Wylaczone, Uzyj /Panel [Gracz]

/sprawdz <gracz>
/cheaty <gracz>
/czysty <gracz>
/brakwspolpracy <gracz>
/ustawsprawdzarke ( ustawia miejsce sprawdzarki )
/setsprawdzaniespawn ( ustawia polozenie spawna po wpisaniu /czysty)
/logout ( mozna  nadac bana za pomoza komendy lub gdy osoba wyjdzie z servera automatyczxnie dostanie bana)
/panel <gracz>


# Co Posiada Ten Plugin?

- Posiada Panel
- Wszystkie komendy potrzebne takie jak: /sprawdz /czysty
- Pobiera Flagi Kazdego Anty-Cheata Z Konsoli I Mozna Zobaczyc Je W /Panel <gracz>

# Panel

Mozna:

- Cheaty
- Czysty
- Przyznanie
- Brak WspolPracy


# Jak Ustawic Permsy:
- sprawdz.admin: ( admin )
- default: op ( Default Operator Servera)

# Jaka Range Stworzyc? -
- Stworz Range Sprawdzany,
- Jesli Masz Default Range Jako Gracz Zostaw, Jesli Masz Gracz Jako Inna Range Ustaw Default Zamiast Rangi Swojej ( Gracz) Lub Skontaktuj Sie Z Developerem Tego Pluginu.
